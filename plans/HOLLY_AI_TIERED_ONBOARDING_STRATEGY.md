## 📊 DATABASE SCHEMA ADDITIONS

### User Mode Selection

```prisma
model UserSettings {
  id          String   @id @default(cuid())
  userId      String   @unique
  user        User     @relation(fields: [userId], references: [id])
  
  // Mode selection
  primaryMode      String   @default("general") // general, music, dev, all-access
  modeConfiguredAt DateTime?
  
  // Mode-specific preferences
  musicRoles       String[] @default([]) // artist, producer, ar, manager, songwriter, label
  musicGenres      String[] @default([]) // pop, hip-hop, r&b, etc.
  
  devRoles         String[] @default([]) // frontend, backend, fullstack, devops, founder
  devLanguages     String[] @default([]) // typescript, python, etc.
  
  // Feature toggles based on mode
  enabledFeatures  Json     @default("{}")
  
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  @@map("user_settings")
}
```
