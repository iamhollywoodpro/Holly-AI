Overview

Welcome to the HOLLY 3.0 API documentation. This comprehensive API provides access to four core engines: Creative, Analytics, Security, and Orchestration. All endpoints use RESTful conventions and return JSON responses.

Base URL

https://api.holly.ai/v1

Quick Start

curl -X GET "https://api.holly.ai/v1/creative/images" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"

Authentication

All API requests require authentication using Bearer tokens. Include your API key in the Authorization header of each request.

Authorization: Bearer YOUR_API_KEY

Getting Your API Key

  1. Sign in to the HOLLY dashboard
  2. Navigate to Settings â†’ API Keys
  3. Click "Generate New Key"
  4. Copy and securely store your key

Creative Engine API

Generate and manage AI-powered creative assets including images, content, and templates.

POST /creative/image/generate

Generate AI-powered images from text prompts.

Request Body

Parameter Type Required Description
prompt string Required Text description of the image to generate
model string Optional AI model to use (default: "dalle-3")
width integer Optional Image width in pixels (default: 1024)
height integer Optional Image height in pixels (default: 1024)

Example Request

{ "prompt": "A futuristic cityscape at sunset", "model": "dalle-3", "width": 1024, "height": 1024 }

Example Response

{ "jobId": "img_abc123", "status": "processing", "estimatedTime": 15 }
GET /creative/images

Retrieve all generated images.

Query Parameters

Parameter Type Description
page integer Page number (default: 1)
limit integer Items per page (default: 20, max: 100)
POST /creative/content/generate

Generate AI-powered text content.

Request Body

Parameter Type Required Description
topic string Required Content topic or subject
type string Required Content type: "article", "social", "email"
tone string Optional Writing tone: "professional", "casual", "formal"
GET /creative/assets

Retrieve all creative assets (images, content, templates).

DELETE /creative/asset/{id}

Delete a specific creative asset.

Analytics API

Access performance metrics, reports, and insights.

GET /analytics/metrics

Retrieve all performance metrics.

Query Parameters

Parameter Type Description
timeRange string "24h", "7d", "30d", "90d" (default: "7d")
category string Filter by category

Example Response

[ { "id": "metric_1", "name": "api_calls", "displayName": "API Calls", "currentValue": 15234, "previousValue": 12890, "changePercent": 18.2, "trend": "up", "unit": "count" } ]
GET /analytics/reports

List all analytics reports.

POST /analytics/report/{id}/run

Execute a specific analytics report.

GET /analytics/insights

Get AI-powered insights and recommendations.

Query Parameters

Parameter Type Description
severity string "high", "medium", "low"

Security API

Monitor security events, run scans, and manage threat detection.

GET /security/events

Retrieve security events and alerts.

Query Parameters

Parameter Type Description
severity string "critical", "high", "medium", "low"
status string "open", "investigating", "resolved"
POST /security/scan

Initiate a comprehensive security scan.

Request Body

Parameter Type Required Description
scanType string Required "vulnerability", "compliance", "full"
target string Optional Specific target to scan
GET /security/scan/{scanId}

Retrieve results from a specific security scan.

Orchestration API

Manage AI agents, workflows, and automation tasks.

GET /orchestration/agents

Retrieve all AI agents and their status.

POST /orchestration/agent

Create a new AI agent.

Request Body

Parameter Type Required Description
name string Required Agent name
type string Required "creative", "analytics", "security", "custom"
config object Optional Agent configuration
POST /orchestration/workflow

Create a new automation workflow.

Request Body

Parameter Type Required Description
name string Required Workflow name
steps array Required Array of workflow steps
trigger object Required Workflow trigger configuration

Example Request

{ "name": "Daily Analytics Report", "steps": [ { "type": "api_call", "endpoint": "/analytics/metrics", "params": { "timeRange": "24h" } }, { "type": "transform_data", "operation": "format_report" }, { "type": "notification", "channel": "email", "recipients": ["team@example.com"] } ], "trigger": { "type": "schedule", "schedule": "0 9 * * *" } }
POST /orchestration/workflow/{id}/execute

Manually execute a workflow.

Webhooks

Receive real-time notifications for events in your HOLLY system.

Supported Events

  • image.generated - Image generation completed
  • content.created - Content creation completed
  • security.alert - Security event detected
  • workflow.completed - Workflow execution finished
  • agent.status_changed - Agent status updated

Webhook Payload Example

{ "event": "image.generated", "timestamp": "2025-12-06T23:30:00Z", "data": { "jobId": "img_abc123", "status": "completed", "imageUrl": "https://cdn.holly.ai/images/abc123.png" } }

Error Codes

The HOLLY API uses standard HTTP status codes and returns detailed error messages.

Code Status Description
200 OK Request successful
201 Created Resource created successfully
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
403 Forbidden Insufficient permissions
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Error Response Format

{ "error": { "code": "invalid_request", "message": "The 'prompt' parameter is required", "details": { "field": "prompt", "issue": "missing_required_field" } } }
