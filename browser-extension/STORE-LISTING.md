## Detailed Description

HOLLY is a self-evolving AI partner that lives in your browser. Access her anytime with a keyboard shortcut.

Features:
- Quick chat with HOLLY from any webpage (Ctrl+Shift+Space)
- Open full HOLLY interface (Ctrl+Shift+H)
- Right-click to send selected text to HOLLY for analysis
- Context-aware assistance based on what you're viewing
- Music generation, code writing, image creation
- Conversation history synced with your HOLLY account

How to use:
1. Install the extension
2. Click the HOLLY icon in your toolbar
3. Enter your HOLLY API URL and API key
4. Start chatting with HOLLY from anywhere

HOLLY can help you with:
- Writing and editing content
- Coding and debugging
- Music production and lyrics
- Image generation
- Research and analysis
- Task management and planning

Your data stays private. The extension only communicates with your configured HOLLY server. No third-party tracking or analytics.
