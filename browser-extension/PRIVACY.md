# HOLLY AI Browser Extension — Privacy Policy

**Last updated:** May 2026

## Overview

HOLLY AI is a self-evolving AI assistant browser extension developed by NEXA Music Group. This privacy policy explains what data we collect, how we use it, and your rights.

## Data We Collect

### Data You Provide
- Chat messages you send to HOLLY
- Files you upload for analysis
- Settings you configure (API keys, preferences)

### Automatically Collected Data
- **None.** The extension does NOT collect browsing history, cookies, or track your activity on other websites.

### Local Storage
The extension uses Chrome's `storage.local` API to save:
- Your API configuration
- Chat history (stored locally only)
- Extension preferences

This data never leaves your machine unless you explicitly send it to the HOLLY API.

## Data Usage

Your chat messages are sent to the HOLLY API server for processing. Messages are used to:
- Generate AI responses
- Maintain conversation context
- Improve response quality within your session

## Data Sharing

We do NOT sell, trade, or share your personal data with third parties. Chat data is processed on our servers and is not shared externally.

## Data Storage & Security

- Chat data is stored on our secure servers
- All communications use HTTPS encryption
- API keys are stored locally in your browser's encrypted storage
- We follow industry-standard security practices

## Third-Party Services

The extension communicates only with:
- `holly.nexamusicgroup.com` (HOLLY API server)
- Your configured HOLLY instance URL

No third-party analytics, advertising, or tracking services are used.

## Your Rights

You can:
- Delete local extension data at any time by removing the extension
- Clear chat history through the extension settings
- Revoke API access by regenerating your API key
- Request data deletion by contacting us

## Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect data from children.
